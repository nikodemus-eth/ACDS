# @acds/worker

Background worker process for recurring ACDS jobs.

## Scripts

- `pnpm --filter @acds/worker run build`
- `pnpm --filter @acds/worker run test`
- `pnpm --filter @acds/worker run typecheck`
- `pnpm --filter @acds/worker run start`

This package is part of the supported MVP runtime surface, but it does not provide the admin UI or GRITS CLI entrypoints directly.
