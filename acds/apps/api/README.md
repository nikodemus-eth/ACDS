# @acds/api

Fastify API surface for ACDS routing, execution, audit, provider, and adaptation operations.

## Scripts

- `pnpm --filter @acds/api run build`
- `pnpm --filter @acds/api run test`
- `pnpm --filter @acds/api run typecheck`
- `pnpm --filter @acds/api run start`

Use this package for the standalone API process. For the canonical operator flow, follow [../../docs/operator/MVP_OPERATOR_GUIDE.md](../../docs/operator/MVP_OPERATOR_GUIDE.md).
