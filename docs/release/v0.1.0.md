# ACDS v0.1.0

## Scope

ACDS `v0.1.0` is the first public MVP release of the Adaptive Cognitive Dispatch System as an ACDS-first product.

Included:

- PostgreSQL-backed API, admin UI, workers, and persistence
- local-first provider posture
- policy-based routing and execution
- audit persistence
- GRITS DB-backed release checks

Secondary:

- Process Swarm remains included as a companion integration example in this workspace

## Known Limitations

- OpenAI, Gemini, and Apple Intelligence integrations are experimental in the MVP release posture
- Adaptive behavior should be treated as observe-first until production policies are validated in the target environment
- Routing decisions are persisted on execution records rather than a dedicated routing decision table

## Upgrade Notes

- Migration `009_grits_and_execution_runtime_alignment.sql` aligns existing execution persistence with the runtime and GRITS contracts
- Existing installations should run all migrations before using DB-backed GRITS release mode
